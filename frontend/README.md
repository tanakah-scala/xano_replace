# xano-replace frontend

Next.js 15 (App Router) + TypeScript + Tailwind CSS によるフロントエンドです。

## ローカル起動

```bash
npm install
npm run dev
# http://localhost:3000
```

## 環境変数

| 変数名 | 説明 | デフォルト |
|--------|------|-----------|
| `NEXT_PUBLIC_API_BASE_URL` | バックエンドAPIのベースURL | `http://localhost:8080` |
